# Assets

Drop the following two files here, named exactly:

- **`cv.pdf`** — your current CV. The CV page (`cv.qmd`) links to it.
- **`headshot.jpg`** — a square photo, roughly 600×600 px, used on the Home page. JPG or PNG both fine; if you use PNG, rename to `headshot.png` and update `index.qmd`'s `image:` field.

Both files are referenced in the site config — drop them in and the next `quarto preview` shows them.
